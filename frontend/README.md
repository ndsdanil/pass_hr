# HR Plus - Frontend

## Описание

Frontend-компонент системы HR Plus, построенный на Next.js. Предоставляет пользовательский интерфейс для загрузки резюме и вакансий, их оптимизации с использованием искусственного интеллекта.

## Технологии

- Next.js 15.1.x
- React 19.x
- TailwindCSS
- Axios для HTTP-запросов
- Framer Motion для анимаций
- PDF.js для работы с PDF-файлами
- Zustand для управления состоянием

## Переменные окружения

Для корректной работы фронтенда необходимо настроить следующие переменные окружения:

```
# URL до API бэкенда
NEXT_PUBLIC_API_URL=http://backend:8000/api
```

В режиме разработки локально используйте:
```
NEXT_PUBLIC_API_URL=http://localhost:8000/api
```

## Установка и запуск

### Локальная разработка

1. Установите зависимости:
```bash
npm install
```

2. Запустите сервер разработки:
```bash
npm run dev
```

Приложение будет доступно по адресу: [http://localhost:3000](http://localhost:3000)

### Сборка для production

```bash
npm run build
npm start
```

### Запуск с использованием Docker

Вы можете запустить фронтенд как часть всей системы с использованием Docker Compose:

```bash
# В корневой директории проекта
cd infra
docker-compose up -d
```

Или собрать и запустить только образ фронтенда:

```bash
docker build -t hr-plus-frontend .
docker run -p 3000:3000 -e NEXT_PUBLIC_API_URL=http://localhost:8000/api hr-plus-frontend
```

## Структура проекта

- `src/` - исходный код приложения
  - `app/` - страницы и маршруты Next.js
  - `components/` - React-компоненты
  - `lib/` - утилиты и вспомогательные функции
  - `store/` - Zustand-хранилища для управления состоянием
  - `styles/` - глобальные стили
- `public/` - статические файлы

## Особенности разработки

- Используется Tailwind CSS для стилизации компонентов
- Компоненты UI из библиотеки Shadcn UI
- Реализована загрузка и отображение PDF-файлов
- Форматирование и проверка кода с помощью ESLint

## Интеграция с бэкендом

Frontend взаимодействует с бэкендом через REST API. Основные эндпоинты:

- `/api/auth/*` - аутентификация и управление пользователями
- `/api/resumes/*` - работа с резюме
- `/api/vacancies/*` - работа с вакансиями
- `/api/tuning/*` - управление процессом оптимизации
